What is readify.me?

readify.me provides a safe, easy, open-source solution that allows you to read articles in without annoying ads and popups.

How do I use it?

It's incredibly easy! Simply visit any article on a modern web browser and place readify.me/ before the URL.

Example:
readify.me/https://www.bbc.co.uk/news/newsbeat-65791039

Chrome user?

Looking for an even easier way to toggle between an article and readify.me? Install the readify.me extension on the Chrome Web Store.

Problem?
